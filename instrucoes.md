# Descrição da Tarefa – LLM - Chatbot Educacional em Python

--- 

## Título da Atividade

Desenvolvimento de um Chatbot Educacional utilizando LLM (Large Language Model) em Python.

---

## Objetivo da Tarefa

Criar um chatbot capaz de responder perguntas de forma clara e didática sobre um determinado tema educacional, utilizando um modelo de linguagem integrado ao Python.

---

## Descrição Detalhada

1. Desenvolver um script em Python que:
   - Permita interação no terminal entre usuário e chatbot.
   - Utilize um modelo LLM (via API da OpenAI ou outro disponível).
   - Responda perguntas com linguagem simples e objetiva, adequada para ensino.
2. Personalizações esperadas:
   - Definir contexto educacional (ex.: Biologia, Matemática, História, Física).
   - Garantir respostas curtas e didáticas.
   - Permitir que o usuário encerre a conversa digitando um comando (“sair”).

---

## Requisitos Mínimos

- Código funcional e bem estruturado.
- Respostas coerentes com o contexto definido pelo aluno.
- Comentários explicando as principais partes do código

---

## Desafios Extras (Opcional)

- Criar interface web usando Flask ou Streamlit.
- Permitir leitura de um arquivo .txt ou .pdf como base de conhecimento.
- Adicionar modo quiz, onde o chatbot faz perguntas ao usuário.

---

## Critérios de Avaliação

- Funcionamento correto do chatbot – **40%**
- Clareza e coerência das respostas – **30%**
- Organização e comentários do código – **20%**
- Criatividade e recursos extras – **10%**
